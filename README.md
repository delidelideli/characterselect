# Embers of Choice | Character Selection Prototype

A cinematic, high-fidelity character selection screen inspired by the "Dark Souls" aesthetic.

## Features
- **Living Background:** Radial gradients, flickering firelight, and drifting ember particles.
- **Weighty Interaction:** Responsive 12-column grid with smooth, cubic-bezier transitions.
- **Glow Beam Borders:** Rotating conic-gradient animations that pulse with heat.
- **The Stage:** Detailed lore and stat blocks for 5 unique classes (Warrior, Sorcerer, Lancer, Cleric, Bandit).
- **Cinematic Typewriter:** Runic-to-text transitions for character lore reveal.
- **"Prepare to Die" Screen:** A full-screen atmospheric final confirmation popup.

## Technical Details
- **Architecture:** Vanilla HTML5 / CSS3 / JavaScript (ES6+).
- **Styling:** Custom CSS with advanced selectors (`:has()`, `@property`) and hardware-accelerated animations.
- **Typography:** Google Fonts (Cinzel Decorative, Cinzel, EB Garamond).

## Project Structure
- `index.html`: Main layout and content.
- `style.css`: All cinematic styling and animations.
- `script.js`: Typewriter logic, particle systems, and grid interaction.
- `DesignDoc_Final.md`: Comprehensive design specifications and visual rules.
- `AI/activity_log.md`: Detailed history of implementation and refinements.
- `Images/`: High-resolution character portraits and renders.
